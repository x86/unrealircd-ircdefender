# Introduction #

The IRC Defender works for now only on UnrealIRCd 3.x protocol.<br />
We making the defender larger and implenting more irc support!


# Build #

Debian Etch 4.0:
  * svn checkout http://unrealircd-ircdefender.googlecode.com/svn/trunk/
  * cd irc\_defender
  * make
  * nano defender.conf
  * ./defender
